## 🧠 Project Overview

Buy Now, Pay Later (BNPL) services like Affirm, Klarna, and Afterpay 
have transformed consumer credit — enabling zero-interest installment 
purchases at checkout with minimal underwriting. But with $70 billion 
in transactions in 2025, 41% of users making late payments, and 45% 
of loans going to deep subprime borrowers, the financial risk is real 
and largely invisible.

This project builds a full data science pipeline to:
- **Model** BNPL-related consumer default risk
- **Visualize** financial stress trends across companies, regions, 
  and demographics
- **Monitor** live macroeconomic indicators linked to consumer 
  financial health
- **Deploy** an interactive R Shiny dashboard accessible to the public

---

## ❓ The Central Question

> Can we build a data-driven tool to model, predict, and visualize 
> BNPL-related financial stress and default risk in real time?
